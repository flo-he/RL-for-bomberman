This folder contains our final agent. 

We did not include the experience buffer, since it is to big to upload (4.4GB)

If one wants to see the tensorboard graph, open console in this folder and call
"tensorboard --logdir=./tb/Duel_DDQN/"
and copy the url in
"TensorBoard 1.13.1 at http://<DEVICE_NAME>:<PORT>"
found in the output of the console into your browser.
